# 第二章：机器学习基础

## 2.1 机器学习概述

机器学习（Machine Learning, ML）是人工智能的核心子领域，研究如何让计算机系统利用数据来改善性能。Arthur Samuel 在1959年将其定义为"使计算机无需明确编程即可学习的研究领域"。

Tom Mitchell 在1997年给出了更形式化的定义：**对于某类任务 T 和性能度量 P，一个计算机程序被认为能从经验 E 中学习，如果它在任务 T 中的性能（由 P 衡量）随着经验 E 的增加而提高。**

### 2.1.1 机器学习的三大范式

| 范式 | 数据要求 | 目标 | 典型应用 |
|------|----------|------|----------|
| 监督学习 | 有标签数据 (x, y) | 学习 x → y 的映射 | 分类、回归 |
| 无监督学习 | 无标签数据 (x) | 发现数据内在结构 | 聚类、降维 |
| 强化学习 | 环境交互 (state, action, reward) | 最大化累积奖励 | 游戏、机器人控制 |

## 2.2 监督学习

### 2.2.1 线性回归

线性回归是最基础的回归模型，假设目标值与特征之间存在线性关系：

$$y = \mathbf{w}^T\mathbf{x} + b = w_1x_1 + w_2x_2 + \cdots + w_nx_n + b$$

**损失函数（均方误差 MSE）**：

$$L(\mathbf{w}, b) = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2$$

**优化方法**：

- **正规方程**：$\mathbf{w} = (X^TX)^{-1}X^Ty$（解析解，适合小数据）
- **梯度下降**：迭代更新，适合大数据集
  - 批量梯度下降（BGD）：每次使用全部数据
  - 随机梯度下降（SGD）：每次使用一个样本
  - 小批量梯度下降（Mini-batch GD）：折中方案

### 2.2.2 逻辑回归

逻辑回归虽然名称含"回归"，但实际是分类模型。它通过 Sigmoid 函数将线性输出映射到 [0,1] 区间：

$$\sigma(z) = \frac{1}{1 + e^{-z}}$$

$$P(y=1|\mathbf{x}) = \sigma(\mathbf{w}^T\mathbf{x} + b)$$

**损失函数（交叉熵）**：

$$L = -\frac{1}{n}\sum_{i=1}^{n}[y_i\log(\hat{y}_i) + (1-y_i)\log(1-\hat{y}_i)]$$

### 2.2.3 决策树

决策树通过递归地选择最优特征进行分裂，构建树形结构。

**核心概念**：

- **信息熵**：衡量数据的不确定性，$H(D) = -\sum_{k=1}^{K}p_k\log_2 p_k$
- **信息增益**：划分前后的熵减，$IG(D, A) = H(D) - H(D|A)$
- **基尼指数**：$Gini(D) = 1 - \sum_{k=1}^{K}p_k^2$

**常用算法**：ID3（信息增益）、C4.5（信息增益比）、CART（基尼指数）

### 2.2.4 支持向量机（SVM）

SVM 的核心思想是找到一个能最大化分类间隔的超平面。

**硬间隔 SVM**（线性可分情况）：
$$\min_{\mathbf{w},b} \frac{1}{2}\|\mathbf{w}\|^2 \quad \text{s.t. } y_i(\mathbf{w}^T\mathbf{x}_i + b) \geq 1$$

**软间隔 SVM**（线性不可分情况）：
引入松弛变量 $\xi_i \geq 0$ 和惩罚参数 $C$：
$$\min_{\mathbf{w},b,\xi} \frac{1}{2}\|\mathbf{w}\|^2 + C\sum_{i=1}^{n}\xi_i$$

**核技巧**：通过核函数将低维数据映射到高维空间，解决非线性问题：
- 线性核：$K(x, x') = x^T x'$
- 多项式核：$K(x, x') = (x^T x' + c)^d$
- RBF 核：$K(x, x') = \exp(-\gamma\|x - x'\|^2)$

## 2.3 无监督学习

### 2.3.1 K-Means 聚类

算法步骤：
1. 随机选择 K 个初始聚类中心
2. 将每个样本分配到最近的聚类中心
3. 重新计算每个聚类的中心（均值）
4. 重复步骤 2-3 直到收敛

**目标函数**：最小化簇内平方和（Inertia）
$$\sum_{i=1}^{K}\sum_{x \in C_i} \|x - \mu_i\|^2$$

### 2.3.2 主成分分析（PCA）

PCA 通过线性变换将数据投影到低维空间，同时保留尽可能多的方差信息。

步骤：
1. 数据中心化（减去均值）
2. 计算协方差矩阵 $C = \frac{1}{n-1}X^TX$
3. 对协方差矩阵进行特征值分解
4. 选取最大的 k 个特征值对应的特征向量组成投影矩阵

## 2.4 模型评估与选择

### 2.4.1 评估指标

**分类问题**：

| 指标 | 公式 | 含义 |
|------|------|------|
| 准确率 (Accuracy) | $(TP+TN)/(TP+TN+FP+FN)$ | 预测正确的比例 |
| 精确率 (Precision) | $TP/(TP+FP)$ | 预测为正的样本中真正为正的比例 |
| 召回率 (Recall) | $TP/(TP+FN)$ | 实际为正的样本中被正确识别的比例 |
| F1-Score | $2 \times P \times R / (P + R)$ | 精确率和召回率的调和平均 |

**回归问题**：

- MAE (Mean Absolute Error): $\frac{1}{n}\sum|y_i - \hat{y}_i|$
- MSE (Mean Squared Error): $\frac{1}{n}\sum(y_i - \hat{y}_i)^2$
- RMSE: $\sqrt{MSE}$
- $R^2$: $1 - \frac{\sum(y_i - \hat{y}_i)^2}{\sum(y_i - \bar{y})^2}$

### 2.4.2 交叉验证

- **Holdout 验证**：简单划分训练集/验证集/测试集
- **K 折交叉验证**：将数据分为 K 份，轮流使用 1 份验证、K-1 份训练
- **留一法（LOOCV）**：K 折的特例，K = n

### 2.4.3 偏差-方差权衡

- **偏差（Bias）**：模型预测的期望与真实值的差异，反映欠拟合
- **方差（Variance）**：模型对不同训练集的敏感度，反映过拟合
- **总误差 = 偏差² + 方差 + 不可约误差**

## 2.5 过拟合与正则化

### 2.5.1 过拟合的解决方法

- 增加训练数据
- 降低模型复杂度
- 正则化
- Dropout（神经网络）
- 早停（Early Stopping）
- 数据增强

### 2.5.2 L1 与 L2 正则化

- **L1 正则化（Lasso）**：$L = L_{original} + \lambda\sum|w_i|$，产生稀疏解
- **L2 正则化（Ridge）**：$L = L_{original} + \lambda\sum w_i^2$，权重衰减
