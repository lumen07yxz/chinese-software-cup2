# 系统设计说明书

## 1. 总体架构

```
前端展示层 (React + Tailwind) 
    ↕ REST API + SSE
API 网关层 (FastAPI + WebSocket)
    ↕
多智能体协作层 (CrewAI 编排)
    ↕
知识服务层 (ChromaDB RAG)
    ↕
基础设施层 (PostgreSQL + Redis + MinIO)
```

## 2. 技术选型

| 层级 | 技术 | 选型理由 |
|------|------|----------|
| 前端 | React 18 + TypeScript + Tailwind CSS + Vite | 组件化、类型安全、快速构建 |
| API | FastAPI + SSE | 异步高性能、原生流式支持 |
| 多智能体 | CrewAI | 角色化编排，轻量级，适合快速开发 |
| LLM | 科大讯飞星火 Spark X2 Flash | 赛题要求，API 成熟 |
| 向量库 | ChromaDB | 嵌入式部署，无需额外服务 |
| 数据库 | PostgreSQL 16 + SQLAlchemy | 成熟稳定，JSONB 支持 |
| 消息队列 | Redis | 会话缓存、状态管理 |
| 文件存储 | MinIO | S3兼容，本地部署 |

## 3. 多智能体协同框架

采用 CrewAI Sequential + Hierarchical 混合模式：

- **ProfileAgent**：画像构建师 - 对话引导、特征抽取
- **OrchestratorAgent**：资源设计总监 - 需求分析、任务拆解
- **DocAgent**：课程文档专家 - Markdown 讲义生成
- **MindMapAgent**：知识架构师 - 思维导图生成
- **QuizAgent**：题库设计师 - 多题型生成
- **VideoAgent**：多媒体编剧 - 视频脚本生成
- **CodeAgent**：实操导师 - 代码案例生成
- **PathAgent**：学习规划师 - 路径规划
- **TutorAgent**：答疑导师 - 多模态解答
- **AssessmentAgent**：评估分析师 - 效果评估

### 协作流程

```
用户输入 → ProfileAgent (画像构建/更新)
              ↓
     OrchestratorAgent (需求拆解)
              ↓
  ┌───────────┼───────────┬───────────┬───────────┐
  ↓           ↓           ↓           ↓           ↓
DocAgent  MindMapAgent QuizAgent VideoAgent CodeAgent
  └───────────┼───────────┴───────────┴───────────┘
              ↓
     PathAgent (路径规划 + 资源整理)
              ↓
     用户学习 + TutorAgent(答疑) + AssessmentAgent(评估)
              ↓ (反馈循环)
     ProfileAgent(画像更新) → PathAgent(路径调整)
```

## 4. 数据流设计

### 对话画像流程
用户输入 → SSE流式传输 → 星火 API 调用 → 特征抽取 → 画像存储(PostgreSQL)

### 资源生成流程
选择资源类型 → RAG检索相关章节 → 构建 prompt → 星火 API 流式生成 → 内容安全过滤 → 存储 + 返回

## 5. UI 设计

- 配色：暖灰/米白基底 + 深墨绿主色 + 琥珀点缀
- 拒绝蓝紫渐变，使用纯色块 + 细微边框分层
- 少量知识元素装饰（课本、神经网络线、数学公式）
- 流式输出 + 骨架屏加载
